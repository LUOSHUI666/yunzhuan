# 云篆 YunZhuan

> 东方美学 · 人格探索  
> 在星图与黄笺之间，遇见本真的自己

---

## 这是什么

云篆是一个**离线即用的东方美学人格体验工具**。打开浏览器，面对旋转的青铜罗盘选择你的性格类型，点击黄笺叩问内心——你会得到一段庄子式的空灵哲思，和一个虚拟的时辰注脚。

没有测评量表，没有标签化解读，没有数据上传。只有星图、罗盘、黄笺、竹简，和一次与自己的短暂对坐。

---

## 如何使用

```
双击 index.html → 浏览器打开 → 入境
```

**零依赖，零安装，零网络。** 任何现代浏览器直接运行。

**PWA 支持：** HTTPS 环境下自动提示"添加到主屏幕"，安装后全屏沉浸、离线可用。

## 体验五幕

| 幕 | 你看见的 | 你在做的 |
|----|---------|---------|
| **入境** | 星图深空缓缓漂移，青铜罗盘分层自转 | 看。什么都不用做 |
| **问道** | 六张黄笺纸上下浮动，墨色字若隐若现 | 左右箭头切换性格类型 |
| **叩心** | 轻点心笺，金色粒子从指尖散开 | 选一张与你当下心境最契合的笺 |
| **观象** | 竹简卡片展开，时辰逐字显现，正文如墨渗纸 | 读。慢慢读 |
| **归真** | 点击别处，一切淡去，罗盘归位 | 道法自然，不必执着 |

## 当前覆盖

- **16 型人格** 全部可选
- **4 种类型**（逍遥散人 / 弈天者 / 渡人舟 / 逐风客）各有完整文案
- **6 张心笺**（迷途 · 坎 · 执 · 散 · 渊 · 生）× 每种 2 条解读
- **48 组**正文 + 时辰，防重复抽取
- 更多类型的文案持续扩充中

---

## 技术底线

- 单文件 HTML + PWA，零外部依赖
- 纯 CSS 绘制所有图形（无图片、无 SVG、无 Canvas）
- 动画锁定 `transform` + `opacity`，确保 60fps
- 文案数据内嵌 JS 对象，修改无需触碰逻辑
- 三级响应式断点（768/480/380），移动端全适配
- 不收集任何用户数据，完全离线可用

---

## 设计哲学

**做减法。**  
用户说"土"→ 去掉假质感层。  
用户说"假"→ 用自然纹理。  
用户说"糙"→ 回到材料本身。

云篆不是界面，是一个仪式空间。每一个像素都应该让人觉得这是木头、纸张、铜器、墨迹——而不是 CSS。
