# 云篆 YunZhuan

> 道门智慧 · MBTI 顾问  
> 以符箓美学，问本命之道

---

## 这是什么

云篆是一个**离线即用的道家仪式化 MBTI 工具**。打开浏览器，面对旋转的青铜罗盘选择你的本命类型，点击符箓令牌启卦——你会得到一段庄子式的空灵哲思，和一个虚拟的问道时辰。

没有测评量表，没有标签化解读，没有数据上传。只有星图、罗盘、符纸、竹简，和一次与自己的短暂对坐。

---

## 如何使用

```
双击 index.html → 浏览器打开 → 入境
```

**零依赖，零安装，零网络。** 任何现代浏览器（Chrome / Edge / Safari）直接运行。

## 仪式五幕

| 幕 | 你看见的 | 你在做的 |
|----|---------|---------|
| **入境** | 星图深空缓缓漂移，青铜罗盘分层自转 | 看。什么都不用做 |
| **问道** | 六张黄符纸令牌上下浮动，朱砂字若隐若现 | 左右箭头切换本命类型，或直接键盘 ← → |
| **启卦** | 点击令牌，金色粒子从指尖爆开 | 选一张与你当下心境最契合的符 |
| **观象** | 竹简卡片展开，时辰逐字显现，正文如墨渗纸 | 读。慢慢读 |
| **归真** | 点击别处，一切淡去，罗盘归位 | 道法自然，不必执着 |

## 当前覆盖

- **16 型 MBTI** 全部可选
- **4 类型**（INFP / INTJ / INFJ / ENFP）各有完整文案
- **6 张令牌**（迷途 · 坎 · 执 · 散 · 渊 · 生）× 每种 2 条解读
- **48 组**正文 + 时辰，防重复抽取
- 更多类型的文案持续扩充中

---

## 技术底线

- 单文件 HTML，零外部依赖
- 纯 CSS 绘制所有图形（无图片、无 SVG、无 Canvas）
- 动画锁定 `transform` + `opacity`，确保 60fps
- 文案数据内嵌 JS 对象，修改无需触碰逻辑
- 不使用纯白 `#FFF`、不使用圆角、不使用 Material Design 等现代 UI 痕迹

## 设计哲学

**做减法。**  
用户说"土"→ 去掉假质感层。  
用户说"假"→ 用自然纹理。  
用户说"糙"→ 回到材料本身。

云篆不是界面，是一个仪式空间。每一个像素都应该让人觉得这是木头、纸张、铜器、墨迹——而不是 CSS。
